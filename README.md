# AF 件数予測＆プランニングツール

Streamlitで実績CSVとCPNマスタを読み込み、媒体別の件数予測、松竹梅プラン、最適プランを作成します。

## 起動方法

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 入力ファイル

### 1. 実績CSV
必須列:

- 成果発生日時
- パートナーサイト名
- 件数
- 報酬額
- 商品ID

文字コードは UTF-8 または Shift-JIS（CP932）に対応しています。

### 2. CPNマスタ（Excel）
必須列:

- 日付
- CPN名

同じ日付が複数行ある場合は、下側の行を採用します。

## GitHubへの登録について

実績CSV・CPNマスタなどのデータファイルはリポジトリに含めません。アプリ起動後、画面から都度アップロードしてください。`.gitignore` でCSV・Excelファイルを除外しています。

## 出力

画面上で「松竹梅」「最適プラン」を確認でき、`af_plan.xlsx` をダウンロードできます。

## 注意点

- 月初フラグは現在「1〜4日」、月末フラグは「28〜31日」の暦日判定です。厳密な営業日判定ではありません。
- 予測ロジックの固定係数は `config/constants.py` および `app.py` 内にあります。
